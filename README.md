# Expense Tracker

A simple full-stack web application to track daily expenses with categories and visual charts.

 Features

- Add daily expenses
- Categorize expenses (Food, Travel, Shopping, etc.)
- View total spending
- Delete expenses
- Simple and clean UI
- Backend API with database storage

 Tech Stack
 Frontend:
- HTML
- CSS
- JavaScript

 Backend:
- Node.js
- Express.js

 Database:
- MongoDB (Atlas)

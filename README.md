# Expense Tracker

A simple full-stack web application to track daily expenses with categories and visual charts.

 Features

- Add daily expenses
- Categorize expenses (Food, Travel, Shopping, etc.)
- View total spending
- Delete expenses
- Simple and clean UI
- Backend API with database storage

  How to Run the Project

1. Clone the repository
2. Open terminal inside backend folder
3. Install dependencies: npm install
4. Create a `.env` file inside backend folder and add:
   MONGO_URI=your_mongodb_connection_string PORT=5000
5. Run backend:  node server.js
6. Open frontend/index.html in browser

 Tech Stack

 Frontend:
- HTML
- CSS
- JavaScript
 Backend:
- Node.js
- Express.js
 Database:
- MongoDB (Atlas)
